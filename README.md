## Overview
The Amazon Prime Video Dashboard provides a comprehensive analysis of movies and TV shows available on the platform. This project was developed using Power BI to transform raw data into meaningful insights through interactive visualizations. The dashboard helps in understanding content distribution, genre trends, ratings, and growth of shows over time, enabling better data-driven insights.

## Data Sources
- Data: Amazon Prime Video dataset (movies & TV shows)
- Tools: Power BI for data visualization and dashboard creation

---

## Key Metrics
- Total Titles: Total number of shows available on the platform  
- Total Ratings: Variety of content ratings (e.g., PG, 13+, 18+)  
- Total Genres: Number of unique genres available  
- Total Directors: Number of directors contributing to content  
- Start Date: Earliest release year in dataset  
- End Date: Latest release year in dataset  

---

## Segmented Data Analysis

### Content Analysis
- Rating Distribution: Number of shows across different age ratings (13+, 16+, PG, etc.)
- Genre Analysis: Most popular genres based on number of shows
- Movies vs TV Shows: Proportion of movies and TV shows available

### Time-Based Analysis
- Release Year Trend: Growth of content over time (1920–2021)
- Identification of peak content addition periods

### Geographic Analysis
- Shows by Country: Distribution of content across different countries using map visualization

---

## Features
- Interactive dashboard with dynamic filters and slicers  
- Drill-down capabilities for deeper data exploration  
- Clean and user-friendly UI design  
- Multiple visualizations for effective storytelling  

---

## Visualizations Used
- KPI Cards (Total Titles, Ratings, Genres, Directors)
- Bar Charts (Rating distribution, Genre analysis)
- Donut Chart (Movies vs TV Shows)
- Line Chart (Content trend over years)
- Map Visualization (Shows by country)

---

## Insights & Usage
This dashboard provides valuable insights for content analysts, streaming platforms, and business stakeholders. It helps in understanding viewer-targeted content, identifying popular genres, tracking content growth, and analyzing global distribution patterns. These insights can support strategic decisions in content production and acquisition.
